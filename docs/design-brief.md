# デザイン方針（Design Brief）

## ムード

- 静謐、寡黙、品
- 青みの黒を基調に、アクセントは深い青を少量
- 余白で語る。装飾で語らない

## タイポグラフィ

- システムフォント優先（読み込みゼロ、速い）
- 行間はやや広め、1カラム、最大幅は狭め（読書体験）

## コンポーネント

- ヘッダー：サイト名のみ
- 説明セクション：`description-text`スタイルでサイト説明を複数段落で表示
- 作品一覧：カード化しない。間と文字で見せる
- 無効項目：「準備中」バッジは控えめ
- 作品ページ：タイトル / 公開年月 / 本文 / フッター

## レスポンシブ

- モバイルファースト
- 文章幅の `max-width` を守り、左右の余白を確保
